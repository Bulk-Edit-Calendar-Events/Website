Windows Defender misidentification (May 17, 2021)

What happened?

Somewhere between the evening of May 16th and the morning of May 17th, 2021, the Windows Defender definition of security risks was updated and it misidentified two of the critical libraries used by Bulk Edit Calendar Events as a Trojan virus. As these libraries build the UI for the application, this resulted in users being unable to launch Bulk Edit Calendar Events. Additionally, you may have seen a message like this:

NOTE: since this is an issue with Windows Defender, this applies to Windows users only.

Is the application safe?

First off: this is a misidentification. Bulk Edit Calendar Events and the libraries that it uses are safe and virus-free.

Secondly: no information has been compromised. Any data used by Bulk Edit Calendar Events is entirely safe and all analytic data is entirely anonymized.

What is being done?

Version 6.0 of the application has been updated and re-uploaded. You can download it from here.

I have also submitted the appropriate files to Microsoft and they are working on clearing the false positive from the defender definition both files have been cleared, but Windows Defender may have removed the files from your computer (requiring you to redownload them) or it may still be warning you (requiring you to clear the Windows Defender cache). You can see some status updates on this where I am communicating with the maintainers of those files here: https://github.com/AvaloniaUI/Avalonia/issues/5937

What do I need to do?

Re-download and re-install v6.0 from here.

That's it! You should be good to go!

Please let me know if you have any other feedback or questions by visiting our support page!
